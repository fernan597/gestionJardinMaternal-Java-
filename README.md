# Sistema de Gestión de Jardín Maternal

Un sistema de escritorio desarrollado en Java diseñado para la administración integral de un Jardín Maternal. El proyecto está enfocado en resolver la gestión de alumnos (salitas), legajos, asistencia, tutores y el personal docente, aplicando los principios de la Programación Orientada a Objetos (POO) y garantizando la persistencia de los datos.

---

 Características Principales

* Gestión de Alumnos: Inscripción, asignación a salas por edades..
* Gestión de Personal: Registro de docentes y no docentes .
* Persistencia de Datos:Almacenamiento de la información mediante archivos de texto .txt

---

Tecnologías Utilizadas

* Lenguaje:Java 
* Paradigma:Programación Orientada a Objetos (POO)
* Interfaz de Usuario:Java Swing

---

Estructura del Proyecto

El código sigue una arquitectura limpia y organizada por paquetes para separar las responsabilidades :

Clases que representan las entidades del sistema (`Alumno`, `Tutor`, `Docente`, `Sala`, `Asistencia`).
Componentes de la interfaz gráfica o clases de interacción con el usuario.
Lógica de negocio, validaciones y control de flujo del sistema.
 Manejo de la persistencia y almacenamiento de datos.
